## Architecture

### background/types.ts
Shared types: TaskState (mutable record), TaskStatus (pending/running/completed/error/cancelled),
ManagedSession (subscribe/abort/dispose), SessionFactory (injectable), ConcurrencyConfig,
FallbackConfig, ErrorClassification, providerKeyOf(), isTerminal().

### background/classifier.ts
classifyError(error) → { retryable, reason }. Retryable: 429, 500, 502, 503, 504, timeout messages.
Fatal: 401, 403, context-overflow messages. Unknown → non-retryable (safe default).

### background/fallback.ts
resolveFallbackModel({ agentName, error, currentModel, triedModels, config }) → FallbackResolution | undefined.
Proactive: agent's fallback_models chain from .pi/agents frontmatter or .autodev/config/fallback.json.
Reactive: next untried model from .autodev/config/models.json allowlist.
loadFallbackConfig(projectRoot) merges frontmatter + config-file chains + allowlist.

### background/manager.ts
BackgroundManager class:
- spawn(config) → taskId. Tracks in Map<taskId, TaskState>.
- Concurrency: per-provider-key counter. Max from concurrency.yaml (default 5). Queue overflow.
- Circuit breaker: sliding-window timer (setTimer injectable for fake timers). Reset on every event.
  Trips after staleTimeoutMs (default 180000). On trip: abort session, mark error, notify parent.
  Idempotent — event wins over concurrent trip (no double-abort, no double-notify).
- Parent-wake: onParentWake callback fires on terminal transition.
- Error handling: on retryable error, resolve fallback, abort+dispose current session,
  update model, re-spawn. On fatal error, surface error (no fallback).
- cancel(id) → abort+dispose+mark cancelled.
- dispose() → clear all timers + dispose all sessions.

### background/index.ts
register(pi) — eagerly initializes manager singleton.
getBackgroundManager() — T9 imports this.
resetBackgroundManager(options) — test-only singleton reset.
Re-exports: BackgroundManager, classifyError, resolveFallbackModel, loadFallbackConfig, types.

### test/mocks/pi-session.ts
MockSession class — controllable event emission via emit(). Tracks aborted/disposed.
mockCreateAgentSession() → SessionFactory mock (bun:test mock()).
mockSessionRegistry — array of all sessions created in current test.
resetSessionMocks() — clears registry between tests.
