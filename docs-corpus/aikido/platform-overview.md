## One Unified Platform to Secure Your Software

From code to cloud to runtime, Aikido covers every layer of modern software security.
Start with the module you need, unlock the full platform as you grow.

### Aikido Code
- **Static Code Analysis (SAST):** AI Autofix, IDE integration, custom rules, context-aware severity scoring.
- **Secrets Detection:** Checks your code for leaked and exposed API keys, passwords, certificates, encryption keys.
- **Malware Protection:** Prevent malicious packages from infiltrating your software supply chain. Powered by Aikido Intel.
- **Infrastructure as Code (IaC):** Scans Terraform, CloudFormation & Kubernetes infrastructure-as-code for misconfigurations.
- **Code Quality:** AI-native code quality rules.
- **Open Source Dependencies (SCA):** Reachability analysis, pre-CVE and malware intelligence, SBOMs, AI-assisted fixes.
- **Open Source License Risks:** Monitors your licenses for risks such as dual licensing, restrictive terms, bad reputation.
- **Outdated Software:** Checks if any frameworks & runtimes you are using are no longer maintained.
- **Container images:** Scans your container images for packages with security issues.

### Aikido Cloud
- **Cloud Misconfigurations (CSPM):** Detects cloud and K8s infrastructure risks (misconfigurations, VMs, Container images).
- **Virtual Machine Scanning:** Scans your virtual machines for vulnerable packages, outdated runtimes and risky licenses.
- **Hardened Images:** CVE-free docker images.
- **Container & K8s Runtime Scanning:** (coming soon)

### Aikido Attack
- **AI Pentesting:** Get a real pentest in hours, not weeks. 200+ agents unleashed that outperform humans.
- **Authenticated DAST:** Test if logged in users can break your application or access sensitive data.
- **API Discovery & Fuzzing:** Automatically map out and scan your API for vulnerabilities.
- **Continuous Pentests:** Infinite pentests.
- **Attack Surface:** Attack surface management.

### Aikido Protect
- **Runtime Protection:** In-app firewall for peace of mind. Block critical injection attacks, introduce API rate limiting.
- **Device Protection:** Protect developer devices from supply chain attacks. Block malicious browser extensions, IDE plugins, code libraries.
- **Bot Protection:** Block malicious traffic.

## AI AutoFix

Aikido's AI AutoFix feature suggests and generates code fixes for certain vulnerabilities.
When a flaw is found, the platform can automatically open a pull request with the proposed fix.
Uses best-in-class LLMs (Claude Sonnet) through Amazon AWS Bedrock.
Customer code is never stored or used for training.
